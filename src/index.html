Проверь себя
1. Какой HTML-тег используется для создания ссылки? Правильный ответ: <a>
2. Какие из этих свойств CSS используются для работы с моделью отображения? Правильные ответы: display, position, float
3. Какой CSS-селектор имеет наибольший вес (специфичность)? Правильный ответ: Селектор по ID
4. Какие из этих значений свойства display делают элемент блочным? Правильные ответы: block, flex, grid
5. Какой CSS-метод лучше всего подходит для горизонтального центрирования блочного элемента с известной шириной? Правильный ответ: margin: 0 auto;
